FF14 MakePlace 使用教程

发表于2022-02-09|Game
|浏览量:|评论数:

前言

之前太久没上游戏 S 房炸了，现在买了个公寓想试试装修成摄影棚，预览了半天家具，然后搜索到一个装修神器 —— MakePlace。该工具可以让你在本地进行装修设计，并包含国际服家具（支持过滤），而且支持中文，如果有使用 XIVLauncher 的话可以安装 MakePlace 插件（国际服 / 国服）在游戏内进行导入与导出。

安装

手动安装

国服建议手动安装，不用翻墙，只是需要手动进行更新。

MakePlace 官网 拉到网页中间左右有个下载按钮，约 1GB 左右的压缩包，官网访问困难的也可以从 百度网盘 下载（可能不是最新的）。

建议解压到 MakePlace 文件夹（自己新建），双击 MakePlace.exe 即可启动程序。

使用 itch 安装

国际服可以用 itch 安装（需要翻墙），支持自动更新。

功能

每次打开都会进入默认场景：

右上工具栏

Settings

点击齿轮图标（Settings）打开设置面板，可以看到右下角显示版本 v121 （比官网显示的 Version 122 小，可能是从版本 0 开始计数）。

在其他选项（Others）中将语言设置为中文，默认隐藏国际服家具

控制选项（Controls）用于设置快捷键

画面选项（Graphics）设置画面的大小与质量，第一次运行时可能是全屏模式，可以改为窗口模式

Layout

加载与保存

MakePlace 中保存与加载的文件称为 Layout，实际是 json 文件

双击加载示例文件 Clair de Lune

House Options

房屋选项

  • 房屋大小：公寓、小型房、中型房、大型房、无人岛（真服了这翻译😅）
  • 照明调整：室内灯光、环境光

Furniture List

显示当前已布置的家具

Fixture List

显示当前设置的装潢配件

Go Outside / Go Inside

除公寓和无人岛之外，都可以进入庭院，或从庭院进入房屋，在庭院可以看到额外的房屋设置

Online Houses

查看大家分享的房屋设计，选中之后可以点击 Visit 进行参观，也可以将当前房屋设计保存到本地 Layout 中。

Online Catalog

查看大家分享的设计，可以直接加载到当前 Layout 中，而不用替换整个房屋。

Rate Places

随机参观并进行评分。

左上工具栏

View & Move Options

默认第一人称 + 飞行模式，如果在高处从飞行模式切换为行走模式会直接落到地面

  • 视角：第一人称 / 第三人称
  • 移动：行走模式 / 飞行模式

Move Mode / Rotation Mode

移动距离，拖动家具时会按照这个距离移动

旋转角度，拖动家具时会按照这个角度旋转

Copy / Delete

复制 / 删除

Undo / Redo

撤销 / 重做

左下工具栏

右下角显示一个选项框，可以过滤 Market Prohibited（禁止交易） 和 Cash Shop only（商城出售）

Fixture

装潢配件

Lighting

照明

Furniture

家具，包含以下分类

  • Cabinet / 橱柜
  • Chair / 椅子
  • Rug / 地毯
  • Shelf / 柜子
  • Table / 桌子
  • Wall-Mounted / 壁挂
  • Others / 其他

Item

物品，包含以下分类

  • Food / 食物
  • Printed / 壁画
  • Replica / 模型
  • Stuffed / 玩偶
  • Trophy / 奖杯
  • Others / 其他

Greenery

植物，包含以下分类

  • Potted / 盆栽
  • Others / 其他

Interative

互动家具

Custom Design

自定义设计

Search

搜索

问题与解决

  1. 没有隐藏国际服家具的选项

    请将语言设置为中文，如果还是没有就重启 MakePlace

  2. 报错 OodleData 相关插件未安装

    旧版本的 MakePlace 没有清理干净，建议使用 Everything 删除所有 MakePlace 相关文件夹后再重新安装。
