文本/图片生成 3D 模型

发表于2025-08-28|更新于2025-10-01|tech
|浏览量:|评论数:

配置

硬件

  • i7-11700@2.50GHz
  • 64GB RAM
  • RTX 3060
    • 12GB

软件

  • Python 3.11.6
  • CUDA 12.8: https://developer.nvidia.com/cuda-12-8-0-download-archive
  • cuDNN 9.12.0：https://developer.nvidia.com/cudnn-downloads
  • TensorRT 10.13.2 GA: https://developer.nvidia.com/tensorrt/download/10x
  • PyTorch 2.8.0+cu128: https://pytorch.org/get-started/locally

模型

本地部署使用，每个模型使用独立的虚拟环境运行。

SF3D

  • Code: Stability-AI/stable-fast-3d
  • HuggingFace: stabilityai/stable-fast-3d

依赖安装报错，把 requirement.txt 里的 texture_baker 和 uv_unwrapper 删除，手动安装

TripoSR

  • Code: VAST-AI-Research/TripoSR
  • HuggingFace: stabilityai/TripoSR

Hunyuan3D-2.0

  • Code: Tencent-Hunyuan/Hunyuan3D-2
  • HuggingFace: tencent/Hunyuan3D-2
  • Live Demo: https://huggingface.co/spaces/tencent/Hunyuan3D-2

直接运行的话路由有问题，需修改 gradio_app.py

运行

启用 text-to-3d 需要加个 patch，在项目根目录新建 fix_lazy_loading.py

更新 gradio_app.py，导入 patch 文件

启用 text-to-3d 需要在运行时添加命令行参数 --enable_t23d

文本转模型相当慢，预计 40min 左右😂

TRELLIS

  • Project: https://microsoft.github.io/TRELLIS
  • Code: microsoft/TRELLIS
  • Live Demo: https://huggingface.co/spaces/trellis-community/TRELLIS

Windows 上直接跑各种问题（Setting Up Trellis on Windows #3），后面直接找了个现成的 IgorAherne/trellis-stable-projectorz installer，暂不支持 text-to-3d

修改 code/gradio_main.py ，绑定到 0.0.0.0

解压后双击运行 run-gradio-fp16.bat，会自动创建虚拟环境。如果需要登录 huggingface 的话进入虚拟环境操作即可。

InstantMesh

  • Code: TencentARC/InstantMesh
  • HuggingFace: TencentARC/InstantMesh

Unique3D

  • Project: https://wukailu.github.io/Unique3D
  • Code: AiuniAI/Unique3D
  • Live Demo: https://huggingface.co/spaces/VIDraft/Unique3D
  1. cuDNN 9.12.0：https://developer.nvidia.com/cudnn-downloads
  2. TensorRT 10.13.2 GA: https://developer.nvidia.com/tensorrt/download/10x
    • Zip File Installation

修改 ./app/gradio_local.py 添加调用参数 allowed_paths，创建并添加临时目录（运行的磁盘，可能需要手动创建）

报错：ImportError: cannot import name 'cached_download' from 'huggingface_hub'

  • 在 .\venv\Lib\site-packages\diffusers\utils\dynamic_modules_utils.py 的 import 中删除 cached_download 导入

报错：显存不足（暂时没管）

requirements.txt

Stable-Zero123

  • Project: https://zero123.cs.columbia.edu
  • Code: cvlab-columbia/zero123
  • HuggingFace: stabilityai/stable-zero123

需要通过 threestudio 使用

添加环境变量

直接跑预估 68h，trainer.max_steps 配置为 600，改成 5 只需要半小时

但是导出模型报错，暂未解决（太难用了）

Hunyuan3D-2.1

  • Code: Tencent-Hunyuan/Hunyuan3D-2.1
  • HuggingFace: tencent/Hunyuan3D-2.1
  • Live Demo: https://huggingface.co/spaces/tencent/Hunyuan3D-2.1

如果报错找不到 torch，则需要新建 ./hy3dpaint/custom_rasterizer/pyproject.toml

同样 gradio_app.py 里的路径需要修改

运行

Texture 生成没有出结果电脑挂掉了……
