|浏览量:|评论数:

Ubuntu Server 20.04 安装 Python3 和 pip

创建虚拟环境

创建一个名为 jupyterenv 的虚拟环境

安装 Jupyter Notebook

在虚拟环境中安装 jupyter

运行 Jupyter Notebook

在虚拟环境中运行 jupyter

访问 Jupyter Notebook

本地浏览器输入服务器 ip 地址访问：ip:8888 ，要求输入运行时自动生成的 token 。

另外，为了让 Jupyter Notebook 在后台运行，建议使用 tmux 。
