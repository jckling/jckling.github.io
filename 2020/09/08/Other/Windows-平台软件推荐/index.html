Windows 平台软件推荐

发表于2020-09-08|Other
|阅读量:|评论数:

0x00 前言

每当拿到一台新电脑或者重装系统之后，首先要做的就是卸载程序，出厂自带的应用（例如某些影音软件）先给卸载了。然后规划一下应用程序的安装路径，通常不会安装在 C 盘（系统盘），此外，也需要规划应用程序的数据存储路径，同样不会放在系统盘。

例如，磁盘分为了 C 盘和 D 盘，那么就在 D 盘创建以下目录，作用见注释。

以上是本人的看法，接下来的内容是个人推荐的软件清单。

0x01 通讯社交

微信

虽然微信真的难用，而且 PC 微信真的丑，但鉴于亲人、老师、同学等等都在使用，所以还是得用…

手机端和电脑端的消息同步不及时不准确，经常出现“断档”，此外，手机上存储的数据和电脑上存储的数据无法互通整合（自己折腾倒是可以）。

还有自带的迷之浏览器真难用 ↓

腾讯 QQ

小学就开始用了，应用程序越来越臃肿（功能越加越多），对于文件的各种支持比微信好得多，个人最喜欢的功能就是本地表情导入，消息同步也比微信好很多，非会员支持 7 天消息漫游。

腾讯会议

疫情期间用来上课、开会之类的，然后这个软件就留下来了。个人觉得简洁易用非常 OK，不过非要用微信登录这点令我反感x

Telegram

telegram yes!

历史消息同步，输入同步，文件支持友好，提供语音聊天功能，消息记录可以导出备份，账号可以注销。作为一款通讯软件来说，功能强大不冗余，缺点是不能搜索中文消息记录。

Discord

下 GShade 的时候顺便加了一下 Discord 群组，但是基本没用过，最近追 VTuber 感觉国外还挺火的，留着了。

更新：Discord 真好用！！！

飞书

之前小组用了用，感觉是集大成之作，同僚和学妹都很推荐，不过我只用来接收消息（因为没什么协作需求2333）。

0x02 必备工具

CCleaner

清理垃圾，卸载软件，免费版的就够用了，强烈推荐。

火绒安全软件

轻巧的安全软件，（个人）最常用的是弹窗拦截、垃圾清理。

Honeyview

图片查看器，个人认为控制栏有点丑，别的方面都很赞，查看 EXIF 信息，快速预览等等功能，好像还可以预览 Photoshop 文件。

Bandizip

功能完善又轻巧的压缩软件，和 Honeyview 同出处。

PotPlayer

  • 中文版

简洁美观，功能齐全，支持多种视频、字幕文件格式，不弹广告。

ScreenToGif

  • Github 仓库

录制屏幕或摄像 gif 的工具，简洁易用。

Snipaste

功能齐全的截屏工具，简单上手，本文插图都是使用该工具截图。

f.lux

电脑屏幕色温调节，根据时间段调整色温。自带一系列配置方案，也可以自定义，调节之后看电脑再也不刺眼了。

Digital Clock 4

找了很久找到的一款桌面时钟软件，简洁易用。美中不足之处是数字宽度不同会导致肉眼可见的位置变动，所以我是关闭秒数使用的x

TranslucentTB

  • Github 仓库

任务栏模糊/透明工具~

Everything

文件搜索神器，结合 Wox 更上一层楼。

Wox

  • Github 仓库

摆脱鼠标神器，正确的使用方法还在探索中。

Git

使用 Github 必备，而且也可以用于在本地管理文件，非常厉害的工具。

Git LFS

git 的大文件扩展，支持图像、音视频、数据集等数据的上传。

Cmder

除了偶尔输入有错位之类的小毛病，总体来说我比较爱用。

Caesium

完整名称 Caesium Image Compressor，本地图片压缩工具，支持常见的 png、jpg/jpeg、bmp 等图片格式。好用！！

MobaXterm

为什么不早点下载来用！会话保存，还有各种工具，支持透明化，家庭版就挺够用的了，有需求可以找找破解版¿

Beyond Compare 4

文件对比软件，UI 非常友好，30 天试用（按打开的天数记录，不打开不记），破解/序列号请自行搜索~

Fiddler

偶尔用来抓包写爬虫，签到脚本 就是根据抓包结果写的，界面没有想象的那么美观，但是功能足够用了！

VNC Server

远程桌面软件，在被控制主机上打开 VNC Server，然后用另一台主机/iPad/手机打开 VNC Viewer 进行远程操控，家庭版够用。

ToDesk

国产的远程桌面软件，用起来比较流畅，就是 UI 有点土，免费版支持 2 台设备。

0x03 下载工具

迅雷

迅雷下东西还是很可以的（确信），缺点是自带烦人的广告，拦截一下就清爽了。

偶尔还会有 bug 般的下载速度，剩余加速流量不变的 ↓

百度网盘

  • 删除设备和驱动器中的PPS、百度云、360云盘图标

虽然百度网盘比较一(la)般(ji)，但许多资源还是通过这个平台分享的，缺点是慢、文件有挂掉的风险。

人人影视

因为有些影视网站上线的影视剧都是删减版，所以我更倾向于在人人上看，时长可以从豆瓣那里得知，删减个十来分钟我还看个啥x

网易云音乐

不知道什么时候突然有了个“网抑云”的梗，我看评论都当乐子看，不会有人真的全信吧？？？

还有不得不吐槽一下网易云的私人FM，难道是定期重置的？不是推荐很早之前就收藏的歌，就是推荐之前 pass 过的歌，真的很让人迷惑和生气。

暂时还没去发掘别的音乐软件，听歌听得不多所以还是蛮用着。

0x04 学习工具

Microsoft 便签

当作备忘录使用了，pin 在桌面上，够用。

OneNote for Windows 10

个人认为这是 Windows 平台上最好的记笔记软件，OneNote yes!

Typora

markdown 编辑器，实时渲染，可以下载主题或自定义，赏心悦目。

Oracle VM VirtualBox

之前上课的时候用虚拟机做实验，现在还在用虚拟机做实验orz

Resilio Sync

文件同步软件，之前找电子书的时候偶然发现的，使用之后感觉确实好用。电脑、iPad、手机局域网同步电子书，下载之后可以直接在本地浏览和更改。

0x05 游戏（平台）

Steam

买了不少游戏但是并没有打完多少… 下次一定下次一定

Epic

开局一个号，游戏全靠领。

Uplay

没怎么用过其实，我只知道刺客信条系列。

碎碎念：看门狗2呢！为什么是光之子！

Microsoft Solitaire Collection

有一段时间沉迷微软纸牌系列，天天玩，还直播玩，现在就偶尔玩一玩，推荐！

最终幻想 XIV

玩了一年多的游戏，人物、剧情、副本等等都值得称赞，可玩性高。2021.9.24 野队把再生零式都打过了，又开始当咸鱼玩家了，不过想了想指挥官还没缝出来、钓鱼没咋钓、bzy 也没肝…… 还是很多可以玩的，期待更新！

0x06结语

暂时列出这些软件，之后发现其他优秀的软件会更新这篇文章，欢迎推荐。
