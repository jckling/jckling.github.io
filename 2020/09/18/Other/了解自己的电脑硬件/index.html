了解自己的电脑硬件

发表于2020-09-18|Other
|阅读量:|评论数:

课堂趣事

高级操作系统的授课老师真的是全程在讲相声，不过从他的话语中可以知道他写过书、做过小型操作系统、参加顶级会议等等，经历非常丰富。个人感觉老师非常厉害而且也很会讲“相声”，还有就是知识储备丰富，问个问题问倒了大片学生。

上课的时候说到电脑数据备份，然后又提到班上同学拿着电脑，就开始问大家的电脑配置，内存、RAM、ROM、显存、硬盘、寻址空间、CPU 等等。说实话我还真的不了解，拿着我的联想 Y7000P-2019 ，只知道是 i7 + 16G + 1T SSD + RTX 2060 ，然后是 1080p 分辨率还有 144 Hz 刷新率…

老师说他硕士毕业创业开发软件，结果因为软件开发行业不景气就把电脑给卖了，顺便给人开发软件，赚的是卖电脑的钱。边说边笑大家都不了解电脑，这怎么卖电脑。

虽然这些仿佛与操作系统无关，但确实，我不了解我的电脑，计算机组成原理的知识也基本都还给老师了。我还时常羡慕 MBP 那么好看，但是我连自己的笔记本都不了解，用得也不深入，这还换个啥子【确信

那么接下来就该好好学习使用自己的 Windows 电脑，先从了解自己的电脑硬件配置开始吧。

硬件

我的笔记本是联想 Y7000P-2019 ，2019 年底刚换，之前用的是联想小新 Air13 Pro ，记得是 256G 固态和 8G 内存。就那破电脑我还用了三年多，最后闲鱼出掉了:)

可以根据型号直接在网上查找硬件信息，在 中关村在线 查看配置参数；

Windows 自带的 系统信息 和 设备管理器 可以查看硬件配置信息；

使用第三方工具 Speccy 可以查看硬件信息。

主板

  • 制造商：LENOVO
  • 模型：LNVNB161216 (U3E1)
  • 芯片组：Intel HM370
  • 功耗：3 W

芯片组的各种信息直接看 Intel 官方给出的产品规范 Mobile Intel® HM370 Chipset 就好（摊手

处理器（CPU）

Intel 酷睿 i7-9750H

  • 主频：2.60 GHz
    • 即，处理器基本频率
    • 表示处理器晶体管打开和关闭的速率
    • 处理器基本频率是 TDP 定义的操作点
  • 最高睿频：4.50 GHz
    • 最大睿频频率是处理器在采用英特尔® 睿频加速技术，以及英特尔® Thermal Velocity Boost（如果存在）时所能达到的最大单核频率。
  • 核心/线程数：六核心/十二线程
    • 6 个内核，12 个逻辑处理器
    • 内核数是一个硬件术语，它表示单个计算组件（裸芯片或芯片）中的独立中央处理器的数量
    • 线程或执行线程是一个软件术语，指代那些可由单核 CPU 传递或处理的基本有序指令序列
  • 三级缓存：12 MB
    • 即，CPU 高速缓存
  • 总线规格：DMI 8GT/s
    • 即，总线速度
    • 总线是在计算机组件之间或计算机之间传输数据的子系统
      • 前端总线（FSB）：它在 CPU 和内存控制器中枢之间传输数据
      • 直接媒体接口（DMI）：这是计算机主板上英特尔集成内存控制器和英特尔 I/O 控制器中枢之间的点对点互联
      • 快速通道互联（QPI）：这是 CPU 和集成内存控制器之间的点对点互联。
  • 核心架构：Coffee Lake
    • 代号名称
  • 制程工艺：14 nm
    • 光刻是指用于生产集成电路的半导体技术，采用纳米 (nm) 为计算单位，可表示半导体上设计的功能的大小
  • 功耗：45 W
    • 热设计功耗 (TDP) 以瓦特为单位
    • 表示所有活动内核在英特尔定义的高复杂性工作负载下，以基本频率运行时消耗的平均功率
  • 指令集：64 bit
    • 指令集即为微处理器理解并能执行的一套基本命令和指令

频率以千兆赫兹 (GHz) 或每秒十亿次循环计。

处理器信息也直接上 Intel 官方查看规范 英特尔® 酷睿™ i7-9750H 处理器 。

补充

虚拟内存

将硬盘的一部分空间作为内存使用，这部分空间即为虚拟内存。

分页文件是硬盘上的一块区域，Windiows 当作 RAM 使用。

所有驱动器总分页文件大小：3416 MB

  • 右键此电脑-属性-高级系统设置-性能设置-高级-虚拟内存更改

因为我选择了 自动管理所有驱动器的分页文件大小 ，因此虚拟内存的大小是动态变化的，允许的最小值为 16 MB 。

参阅

  • 虚拟内存

寻址空间

64 位架构下，地址线是 46 个，所以最大的物理地址是 2^46 B ，折合 64 TB

参阅

  • Address space
  • 寻址模式
  • 32位系统只能寻址4G空间，64位则是128G，这些是怎么算出来的？

总线

根据关于寻址空间的知乎问题回答中得知，地址总线 46 位，需要参考 Intel® 64 and IA-32 Architectures Software Developer Manuals 。

参阅

  • Bus (computing)
  • 64-bit computing

时钟周期 / 时钟频率

时钟频率（时钟速度） 2.60 GHz ，即每秒钟执行 2600,000,000 个时钟周期，时钟周期为时钟频率的倒数。

参阅

  • Clock rate
  • What is a clock cycle and clock speed?
  • 什么是时钟速度？

总结

捋一捋才发现，哇，原来这么多硬件，这么多参数，然后看得我眼花缭乱，满脑子“这是啥？”的想法。也算是稍微了解了一些电脑硬件配置，不过每个硬件的参数对电脑什么方面的性能有影响，这个倒是没有弄清楚。

看到英特尔架构手册就让我想起了当年上逆向工程的时候，感谢范大让我早早知道和稍微翻阅了一下，内容真的是超级多（然后也没记住）。

计算机组成原理的知识大部分归还给了克敏，个人感觉自己学知识非常“应试”，学完考完就忘，而且我本来就觉得自己健忘，也有可能是心理作用？这可相当不行，老懒熊要开始改坏习惯了【确信
