Medium

源码解析

unsafe-inline 允许使用内联 JavaScript 和 CSS；nonce-source 使用随机数源仅允许特定的内联脚本块。

漏洞利用

同样使用开发者工具查看，unsafe-inline 表示允许使用内联资源nonce-source 表示仅允许特定内联；

直接注入 JavaScript 代码，成功执行。

参阅

  • 内容安全政策
  • CSP: script-src

High

源码解析

只允许本页面加载的 JavaScript 执行，不过这里还接收 include 参数；

点击按钮后生成 script 标签指向 source/jsonp.php?callback=solveSum，并加入 DOM 中；页面执行 JavaScript 代码中的 solveSum 函数，在页面中写入答案。

漏洞利用

点击 Solve the sum 后发现利用了一个参数为 callback 的 .php 脚本执行函数 solveSum ，这个函数刚好在 high.js 中；

这里使用 Burp Suite 抓包修改参数 callback ，绕过成功。

Impossioble

源码解析

self 信任来自站点的同一个源 (不包括其子域名)，并且未设置信任内联元素；

不使用 callback 函数，直接硬编码，避免了注入。
